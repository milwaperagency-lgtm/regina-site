# Регина — для тебя

Сайт-признание в любви. Mobile-first, длинный скролл.

## Блок «Наши даты»

Даты в `index.html` в секции `.dates-section` — замени на свои:
- Первая встреча
- Первое свидание
- Первый поцелуй
- Мы вместе

## Сезонные эффекты

Автоматически показываются в зависимости от месяца:
- **Зима** (дек–фев): снег
- **Весна** (мар–май): лепестки
- **Лето** (июн–авг): лепестки
- **Осень** (сен–ноя): листья

## Конфетти

Появляется при первом открытии сайта в этой сессии браузера.

## Видео

Акцентное видео: `assets/videos/IMG_9808.MOV`. Для стабильной работы во всех браузерах нужен MP4.

- **С ffmpeg:** запусти `convert-video-to-mp4.bat` — создастся `IMG_9808.mp4`, сайт подхватит его автоматически.
- **Без ffmpeg:** конвертируй MOV в MP4 онлайн (например, cloudconvert.com), сохрани как `assets/videos/IMG_9808.mp4`.

## Фотографии (коллаж)

В коллаже используются фото из `assets/images/`: img1.png, img2.png, img4.png, img5.png, img8.png. Последнее фото выводится на всю ширину. Чтобы заменить — подставь свои файлы с теми же именами или измени пути в `index.html` в секции `.gallery-grid`.

## Запуск

Просто открой `index.html` в браузере. Или через локальный сервер:

```
npx serve
```
